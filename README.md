# Snake-game


A fully playable, built directly in the terminal using Python's curses library.

Why I Built This
ive played this gaem before i t was a good project. 

Features
Progressive Difficulty: The game speeds up dynamically as your score increases.
Persistent High Scores: Your best score is automatically saved locally to highscore.txt and loads the next time you play.
Pause Functionality: Hit P to pause, changes nothing pause and unpase freely.


Prerequisites
You need Python 3 installed on your computer.

Bash
python snake.py
Windows
Windows does not include the curses library by default linx and mac do though. You will need to install the official port before running the game:

Bash
pip install windows-curses
python snake.py
🎮 How to Play
Controls
Arrow Keys (↑, ↓, ←, →): Change the snake's direction.
P: Pause / Unpause the game.
Q: Quit the game.

Game Over Screen when you lose lol

Game Over screen:
Press R to instantly restart a new game.
Press Q to quit to your terminal.
